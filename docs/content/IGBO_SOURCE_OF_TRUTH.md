# Igbo Content Source of Truth

## Primary Reference

Okowaokwu Igbo Ụmụaka: Igbo Dictionary for Children
Author: Yvonne Chiọma Mbanefo
Publisher / Platform: Learn Igbo Now
Website: www.LearnIgboNow.com

## Usage Rule

Use this source to verify child-friendly Igbo vocabulary, spelling, tone marks, and English meaning alignment.

Do not copy large sections of the dictionary directly into the app unless permission or license has been confirmed.

## QA Priority

1. Verify every English word maps to the correct Igbo word.
2. Prefer child-friendly, commonly taught vocabulary.
3. Preserve correct Igbo diacritics.
4. Flag dialect differences instead of silently mixing them.
5. Where Central Igbo differs from Ogwashi-Ukwu / Enuani usage, Central Igbo remains the default and variants should be marked separately.
