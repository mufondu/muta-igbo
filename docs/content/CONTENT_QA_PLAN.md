# Mụta Igbo Content QA Plan

Before shipping, every lesson, game, quiz, translator prompt, and review card must be validated for:

1. Correct Igbo word
2. Correct English equivalent
3. Singular/plural accuracy
4. Dialect note if not Central Igbo
5. Picture/icon match
6. Child-friendly wording
7. Audio pronunciation readiness

## App-wide QA rule

No content is considered production-ready until the Igbo, English, visual, and audio all match.

## Known examples to validate

- 100 must be narị or otu narị, not otu
- 9 must be verified for Central Igbo and Anambra variants
- Chicken singular must be ọkụkọ
- Chickens plural must be ụmụ ọkụkọ
- Spelling game must not use paired concepts like Big / Small
- Complete It must only use sentence-compatible vocabulary
- Quiz number options must use number-card format app-wide
