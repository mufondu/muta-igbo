# MekaOps Edu Path System

Mụta Igbo does not use generic worksheet-style level branding.

The app uses a proprietary MekaOps Edu progression system designed to feel intuitive, child-friendly, and ownable.

## Path Map

| Internal ID | MekaOps Edu Path | Lesson Theme |
|---|---|---|
| 7A | Sound Garden | Alphabet & Sounds |
| 6A | Hello Village | Greetings & Phrases |
| 5A | Counting Market | Numbers & Counting |
| 4A | Family Tree | Family & People |
| 3A | Home & Food | Food, home, and daily objects |
| 2A | Nature Walk | Animals & Nature |
| 1A | Word Builder | Grammar & Language |

## Product Rule

Internal IDs may remain in code for routing and progress tracking, but user-facing UI should use MekaOps Edu Path names.
