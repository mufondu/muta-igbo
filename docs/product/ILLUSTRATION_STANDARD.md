# Mụta Igbo Illustration Standard

All lesson visuals should use custom, warm, kid-friendly cartoon illustrations.

## Visual Direction

- Premium children's book style
- Warm, friendly, expressive
- African / Igbo representation where human characters appear
- Clear single-concept images
- No generic emoji as primary lesson art
- No distorted AI/procedural faces
- No adult-looking or romantic symbols for children's content
- One image must map to one concept

## App-wide Rule

If a design improvement is made for one lesson type, it becomes the app-wide pattern.

## QA Rule

Every illustration must pass:

1. Looks child-friendly
2. Clearly matches the English word
3. Clearly matches the Igbo word
4. Works at small card size
5. Does not introduce cultural or visual confusion
