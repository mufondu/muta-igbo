# Mụta Igbo Family Illustration Standard

This is the approved visual direction for Family Members.

## Approved Style

- Premium semi-realistic cartoon portrait
- Igbo / Anambra-inspired attire
- Warm, kid-friendly facial expression
- Rounded-square portrait crop
- Soft village/home background
- Clear family-role distinction
- Readable at 74px lesson-card size

## Approved Family Asset Mapping

| App Role | Required Asset |
|---|---|
| Father / Nna | adult male in red cap |
| Mother / Nne | adult woman in blue headwrap |
| Grandfather / Nna ochie | older man in red cap |
| Grandmother / Nne ochie | older woman in burgundy headwrap |
| Brother / Nwanne nwoke | boy in red cap |
| Sister / Nwanne nwanyị | girl with bow |
| Uncle / Nna nna | adult man in brown cap |
| Aunt / Nne nne | adult woman in colorful headwrap |
| Children / Ụmụ | group image |
| Child / Nwa | young child portrait in Igbo attire |
| Cousin / Ibe | child/young relative portrait, not reused mother/father art |

## Do Not Ship

- Generic emoji as final family art
- Duplicated Mother image for Brother, Sister, Uncle, or Aunt
- Procedural placeholder faces
- Ada / Daughter in the current Family Members section
- Extra cards such as Generation, People, Friend, Guest, or Family in this section

## Product Rule

Family image matching must prioritize the English role first. Igbo words like Nwanne can overlap with Nne, so exact role matching is required before fallback matching.
