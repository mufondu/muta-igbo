# Mụta Friends Onboarding Standard

The child profile name remains the child identity. The selected Mụta Friend is the learning companion.

Rules:
- Render names and interests in React Native text only. Do not bake text into avatar images.
- Use portrait-only character images with centered face and shoulders.
- Keep the onboarding layout scroll-safe on small phones.
- No fixed bottom button overlays on the profile screen.
- Use `ProfileImage` or `MutaFriendAvatar` everywhere a profile avatar appears.
