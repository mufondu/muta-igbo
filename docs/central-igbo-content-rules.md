# Mụta Igbo Central Igbo Content Rules

Mụta Igbo uses Central Igbo as the primary learning path. Ogwashi-Ukwu / Enuani wording is allowed only as a labeled dialect support field.

## Hard rules

1. Do not publish unverified vocabulary.
2. Do not use English one-word mapping when context changes the Igbo word.
3. Prefer child-friendly phrases over isolated dictionary forms for actions.
4. Keep dialect variants out of the primary lesson text unless labeled.
5. All human illustrations must be Igbo-themed and kid-friendly.

## Example

Wrong:

- Drink = Nwa

Correct:

- Drink as a verb = Ịṅụ
- Instruction phrase = Ṅụọ mmiri
- Water = Mmiri
- Child = Nwa

## Review fields

Each vocabulary item should eventually have:

- English
- Central Igbo
- Ogwashi-Ukwu variant, optional
- Part of speech
- Child-friendly phrase
- Example sentence
- Audio file
- Illustration prompt
- Verified flag
- Reviewer
- Notes
