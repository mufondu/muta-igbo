# Mụta Igbo Product Principles

## Mission
Help children grow up understanding, speaking, and loving Igbo through joyful, interactive, culturally authentic learning.

## Build Rules

1. Child first
Every feature must make learning easier, more joyful, or more memorable for a child.

2. Quality over quantity
Five polished games are better than twenty average games.

3. Central Igbo first
Central Igbo is the default. Dialect notes are secondary and must not confuse the main lesson.

4. No unverified vocabulary
Words must come from the verified MekaOps Edu Central Igbo vocabulary list before entering lessons.

5. Polish before expansion
Improve existing lessons, games, sounds, and visuals before adding new feature categories.

6. No feature creep
If a feature does not help a four-year-old learn or help a parent trust the app, it waits.

7. Stop when version 1 solves the problem
Version 1 does not need every idea. It needs to be beautiful, useful, stable, and lovable.
